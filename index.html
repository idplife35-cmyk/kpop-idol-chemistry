FAQ

Answers to common questions about how it works.

How are the names generated?

The generator creates two names using the idol you pick and a style preset.

  • Same-name: a random Korean surname + the idol's Korean name (e.g., Kim + Jungkook).
  • Styled-name: builds a new given name using syllable pools tuned per relationship type.

Results are deterministic: the same inputs produce the same outputs.

What is the chemistry score?

A fun, seeded score between 70â€“100 based on your inputs. It's not a real compatibility measure.

How does the Gender option work?

Male/Female use different syllable pools for styling. Auto tries to infer from your name or the idol.

  • English names: simple lists (e.g., Sophia/Liam) guide the guess.
  • Korean names: the last character heuristic helps choose likely gender.
  • If unknown, the idol's gender is used as a fallback.

Privacy: is anything stored or sent?

Everything runs in your browser. We don't upload your inputs to a server.

  • Only your language preference is saved in localStorage.
Why show both Korean and English names?

This site supports international users. English names are romanized using a simplified mapping.

Can I use this offline? Any requirements?

Use a local or web server; opening the file directly won't work for modules or data fetches.

  • Examples: VS Code Live Server, or python3 -m http.server

Full offline mode isn't supported because the site fetches data files dynamically.

How do I change language?

Use the language selector at the top-right. The page updates instantly and remembers your choice.
