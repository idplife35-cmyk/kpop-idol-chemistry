- 설치 위치: `src/layouts/BaseLayout.astro` L117~L125 (동적 로딩, gtag config)

- 언어 세그먼트: GA4 탐색 보고서에서 `언어 = en` / `언어 = ko` 분리 가능
- 국가 세그먼트: M1 결정한 영어권 우선이므로 `미국 + 영국 + 캐나다 + 호주 + 동남아 영어권` 묶음 저장 권장

### GA 데이터 접근 방식
- **현재 (P0):** Playwright로 GA 홈 화면 접속 → 스크린샷 + DOM 파싱
- **목표 (P1, Week 1 안에):** GA Data API v1 (무료, 서비스 계정 + JSON 키)
  - 셋업 절차:
    1. Google Cloud Console에서 프로젝트 생성 → GA Data API 활성화
    2. 서비스 계정 생성 → JSON 키 다운로드
    3. GA4 속성 관리에서 서비스 계정 이메일에 `뷰어` 권한 부여
    4. `~/project/kpop-idol-chemistry/analytics/ga_client.{ts,py}` 작성
  - 비용: 무료 (할당량: 25,000 토큰/일, 일반 사용엔 충분)
- 사용자 정책 부합: "API가 유료라면 웹페이지로" → GA Data API는 **무료**이므로 API가 정답

### AdSense 데이터 접근
- 현재: AdSense 콘솔 수동 확인 추정
- 목표: AdSense Management API (무료, OAuth 2.0)
- 또는 GA4와 AdSense 연동 → GA4 보고서에서 직접 확인
