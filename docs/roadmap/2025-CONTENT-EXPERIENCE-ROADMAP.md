## 📌 프로젝트 정체성 요약

> **"K-Pop 팬이 좋아하는 아이돌과의 특별한 연결을 만들어주는 인터랙티브 플레이그라운드"**

### 핵심 가치
| 가치 | 설명 | 구현 방법 |
|------|------|-----------|
| **연결 (Connection)** | 팬과 아이돌 사이의 가상 연결고리 | 케미 점수, 관계 타입 |
| **정체성 (Identity)** | 팬덤 안에서의 나만의 아이덴티티 | 한국식 이름, 스테이지 네임 |
| **놀이 (Play)** | 혼자 또는 친구와 즐기는 재미 | VS 모드, 배지, 레벨 시스템 |

### 타겟 사용자
- **Primary**: K-Pop 팬 (13-25세, 글로벌)
- **Secondary**: 팬픽 작가, SNS 콘텐츠 크리에이터
- **Use Cases**: 케미 테스트, 팬픽 캐릭터명, 친구와 대결, 덕질 표현

## 🎨 경험 개선 계획

### UX 개선 항목

| 영역 | 현재 | 목표 | 우선순위 |
|------|------|------|:--------:|
| 첫 화면 로딩 | ~2초 | <1초 | P0 |
| 모바일 터치 영역 | 작음 | 44px+ | P0 |
| 결과 공유 | 텍스트만 | 이미지 카드 | P1 |
| 히스토리 접근성 | 스크롤 필요 | 탭 네비게이션 | P2 |

### 게이미피케이션 확장

#### 현재 배지 시스템
- 레벨 배지 (5개)
- 생성 횟수 배지 (5개)
- 케미 점수 배지 (3개)
- VS 모드 배지 (2개)
- 일일 도전 배지 (2개)
- 컬렉션 배지 (2개)
- 소셜 배지 (2개)

#### 추가 예정 배지
| 카테고리 | 배지명 | 조건 | 레어도 |
|----------|--------|------|--------|
| 그룹 | BTS 마스터 | BTS 전 멤버 컬렉션 | Rare |
| 그룹 | 올그룹 탐험가 | 10개 그룹 이상 시도 | Epic |
| 시즌 | 얼리버드 | 신규 그룹 추가 첫 주 시도 | Uncommon |
| 시즌 | 연말 파티 | 12월에 100회 생성 | Rare |
| 특별 | 천생연분 컬렉터 | 100점 케미 5회 달성 | Legendary |

## 📝 콘텐츠 가이드라인

### 새 페이지 작성 시 필수 요소
1. **SEO**
   - 고유한 메타 description (150-160자)
   - 타겟 키워드 포함
   - FAQPage 스키마 (6-10개)
   - WebApplication 스키마
   - hreflang 태그

2. **UX**
   - 멤버 카드 (클릭 가능)
   - 그룹 배너/로고
   - 즉시 사용 가능한 폼
   - 모바일 최적화

3. **브랜딩**
   - Kitsch 스타일 유지
   - 일관된 색상 팔레트
   - 재미있는 마이크로카피

### 하지 말아야 할 것
- ❌ 인터랙션 없는 정적 정보 페이지
- ❌ K-Pop과 관련 없는 일반 콘텐츠
- ❌ 복잡하고 시간이 오래 걸리는 기능
- ❌ 다른 사이트 콘텐츠 복사
