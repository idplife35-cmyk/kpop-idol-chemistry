## 🗓️ 실행 계획

### Phase 0: 바이럴 트리거 강화 (3-5일) ⭐ 최우선

> **목표**: "공유하고 싶은" 콘텐츠 만들기

#### 0-1. Ship Name 업그레이드

**현재:**
```
Korean: 정유나 (JeongYuna)
English: JK + You = JKYou
```

**변경 후:**
```
💕 Your Ship Name with Jungkook

🎀 Cute Ver: 꾹이의 토끼 (Kookie's Bunny)
🔥 Chaos Ver: 근육토끼즈 (Muscle Bunny Squad)  
📱 Insta Bio: 정국이 내꺼 | JK's Person
🎵 Fandom Tag: #꾹러버 #JKLover
✨ Ship Motto: "운명처럼 만난 우리"
```

**멤버별 스타일:**
| 멤버 | 테마 | 예시 |
|------|------|------|
| 정국 | 토끼/운동 | 꾹이의 토끼 🐰 |
| V | 곰/예술 | 태태의 뮤즈 🐻 |
| 지민 | 모찌/춤 | 지미니의 모찌 🍡 |
| 리사 | 고양이/패션 | 리리의 킹덤 👑 |

#### 0-2. 희귀 케미 시스템

| 등급 | 점수 | 메시지 | 희귀도 |
|------|------|--------|--------|
| 일반 | 70-84% | "좋은 케미네요! 💕" | 60% |
| 높음 | 85-94% | "운명적인 만남! ✨" | 30% |
| 전설 | 95-99% | "🔥 상위 5% 천생연분!" | 9% |
| 신화 | 100% | "💎 0.1% 완벽한 소울메이트!" | 1% |

→ **높은 점수 = 자랑하고 싶음 = 공유**

#### 0-3. 결과 화면 감정 강화

```
┌─────────────────────────────┐
│  💜 LEGENDARY CHEMISTRY 💜  │
│                             │
│    You & Jungkook           │
│         94%                 │
│    ⭐⭐⭐⭐⭐               │
│                             │
│  "상위 3%의 특별한 인연"     │
│                             │
│  🔥 친구에게 자랑하기        │
└─────────────────────────────┘
```

**예상 효과:**
- 공유율: 1% → **5%+**
- 체류시간: +15초

### Phase 2: 페이지 확장 (2주)

> **목표**: SEO 롱테일 확보

#### 2-1. 개별 아이돌 페이지 (차별화 버전)

**URL:** `/bts-name-generator/jungkook/`

**페이지 구성:**
```
1. 멤버 소개 (팬덤 감성)
   - "황금막내 정국"
   - 팬들이 부르는 애칭들
   - MBTI, 좋아하는 것

2. 케미 테스트 (해당 멤버 자동 선택)

3. 정국 전용 Ship Name 스타일
   - 토끼 관련 🐰
   - 운동 관련 💪
   - 노래 관련 🎤

4. "정국 팬들의 평균 케미"
   - "정국 팬 10,000명 중 당신은 상위 12%"

5. FAQ (SEO 최적화)
   - "정국의 본명은?"
   - "정국 MBTI는?"
   - "정국 생일은?"
```

#### 2-2. 블로그 (팬 감성 콘텐츠)

**콘텐츠 방향:**
