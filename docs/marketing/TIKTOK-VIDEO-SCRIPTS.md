## 🎯 TikTok 전략

### 핵심 공식
```
Hook (0-3초) → 과정 (4-10초) → 결과 (11-15초) → CTA (16-20초)
```

### 성공 요소
1. **Hook**: 처음 3초에 시선 사로잡기
2. **Emotion**: 감정 표현 (놀람, 기쁨, 웃음)
3. **Trend**: 인기 음악 + 효과음
4. **Text Overlay**: 자막 필수 (한글 + 영어)
5. **CTA**: 명확한 행동 유도

## 📱 촬영 팁

### 장비
- **필수**: 스마트폰 (최신 모델 추천)
- **권장**: 삼각대 또는 거치대
- **조명**: 자연광 또는 링라이트
- **마이크**: 내장 마이크도 OK (조용한 공간)

### 촬영 설정
```
해상도: 1080p 이상
프레임: 30fps 또는 60fps
비율: 9:16 (세로)
길이: 7-30초 (최적: 15초)
```

### 편집 앱
- **CapCut** (무료, 가장 추천)
- **InShot** (무료)
- **VN** (무료)
- **TikTok 내장 편집** (간단한 경우)

### 편집 요소
1. **텍스트**: 큰 글씨, 대비 높게
2. **음악**: 트렌딩 음악 사용
3. **전환**: 빠르고 역동적
4. **자막**: 한글 + 영어
5. **이모지**: 적절히 활용
